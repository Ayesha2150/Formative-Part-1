# SPCA Cape of Good Hope Website Summative 1

## Overview
This website showcases the SPCA Cape of Good Hope, providing information on adoption, donations, education, services, and contact. It is fully responsive for desktop, tablet, and mobile devices, and uses a consistent color palette of Dark Blue, Orange, and White for a professional yet friendly look.

## Website Structure (HTML Pages)

### Home (index.html)
- Hero image of a dog with compelling call-to-action
- Highlights urgent rescue cases and key stories
- Prominent Donate and Adopt buttons
- Responsive highlights grid showcasing main services

### About Us (about.html)
- Information about the SPCA's mission, vision, and values
- Brief history and role in the community
- Impact statistics with animated numbers
- Team gallery and shelter information

### Adopt (adopt.html)
- Searchable adoption gallery with photos and profiles of animals
- Real-time search functionality to filter pets
- Step-by-step explanation of the adoption process
- Adoption fees and requirements clearly displayed
- Interactive pet cards with modal image viewing

### Donate (donate.html)
- Donation form with name, email, amount, and payment method
- Redirects to a Thank You page after submission
- Enhanced form validation and user feedback

### Thank You (thankyou.html)
- Confirms donation submission with appreciation message
- Includes a "Return to Home" button for easy navigation

### Education (education.html)
- Animal welfare facts, awareness campaigns, and events
- School visits, workshops, and community programs
- Engaging content layout with interactive elements

### Contact (contact.html)
- Comprehensive contact methods grid (phone, email, visit, emergency)
- Professional business contact form with department selection
- Interactive map, location details, and operating hours
- Social media links (Facebook, Instagram, Twitter)
- Emergency contact banner for urgent situations

### Service Enquiry (enquiry.html)
- Service-specific enquiry form for adoption, volunteering, and sponsorship
- Detailed service selection and comprehensive information gathering
- Enhanced validation and success messaging

## Installation & Setup

1. Clone or Download the project files
2. File Structure should maintain:
   - HTML files in root directory
   - CSS files in /CSS/ directory
   - JavaScript files in /JS/ directory
   - Images in /images/ directory
3. Open index.html in any modern web browser
4. No server required - works as static website
